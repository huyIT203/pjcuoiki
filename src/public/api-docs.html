Tài liệu E-commerce API

Tài liệu tham khảo đầy đủ cho tất cả các điểm cuối API

Xác thực Người dùng Sản phẩm Danh mục Đơn hàng Giỏ hàng Đánh giá Quản trị viên Nhà cung cấp Thuộc tính Bộ sưu tập Postman

Xác thực

POST /api/users/register

Đăng ký người dùng mới

Yêu cầu mẫu:
{
  "name": "Nguyễn Văn A",
  "email": "example@email.com",
  "password": "password123",
  "passwordConfirm": "password123"
}
POST /api/users/login

Đăng nhập và lấy access token

Yêu cầu mẫu:
{
  "email": "example@email.com",
  "password": "password123"
}

Quản lý người dùng

GET /api/users/profile

Lấy thông tin người dùng hiện tại (Yêu cầu xác thực)

PUT /api/users/profile

Cập nhật thông tin người dùng (Yêu cầu xác thực)

GET /api/users

Lấy danh sách tất cả người dùng (Chỉ dành cho Admin)

Quản lý sản phẩm

GET /api/products

Lấy danh sách tất cả sản phẩm với phân trang và lọc

Query params: page, limit, sort, price[gte], price[lte], category, brand, rating

GET /api/products/:id

Lấy chi tiết một sản phẩm theo ID

POST /api/products

Tạo sản phẩm mới (Chỉ dành cho Admin)

PUT /api/products/:id

Cập nhật sản phẩm (Chỉ dành cho Admin)

DELETE /api/products/:id

Xóa sản phẩm (Chỉ dành cho Admin)

Quản lý danh mục

GET /api/categories

Lấy danh sách tất cả danh mục

POST /api/categories

Tạo danh mục mới (Chỉ dành cho Admin)

Quản lý đơn hàng

GET /api/orders

Lấy danh sách tất cả đơn hàng (Chỉ dành cho Admin)

GET /api/orders/myorders

Lấy đơn hàng của người dùng hiện tại (Yêu cầu xác thực)

GET /api/orders/:id

Lấy chi tiết đơn hàng theo ID (Yêu cầu xác thực)

POST /api/orders

Tạo đơn hàng mới (Yêu cầu xác thực)

PUT /api/orders/:id/status

Cập nhật trạng thái đơn hàng (Chỉ dành cho Admin)

PUT /api/orders/:id/cancel

Hủy đơn hàng (Yêu cầu xác thực)

Quản lý giỏ hàng

GET /api/cart

Lấy giỏ hàng của người dùng hiện tại (Yêu cầu xác thực)

POST /api/cart

Thêm sản phẩm vào giỏ hàng (Yêu cầu xác thực)

PUT /api/cart/:itemId

Cập nhật số lượng sản phẩm trong giỏ hàng (Yêu cầu xác thực)

DELETE /api/cart/:itemId

Xóa sản phẩm khỏi giỏ hàng (Yêu cầu xác thực)

Quản lý đánh giá

GET /api/products/:id/reviews

Lấy tất cả đánh giá của một sản phẩm

POST /api/products/:id/reviews

Thêm đánh giá cho sản phẩm (Yêu cầu xác thực)

Quản lý Admin

POST /api/users/setup

Tạo admin đầu tiên cho hệ thống (Chỉ dùng khi chưa có admin nào)

Yêu cầu mẫu:
{
  "name": "Admin",
  "email": "admin@example.com",
  "password": "adminpassword123",
  "passwordConfirm": "adminpassword123"
}
POST /api/users/admin

Tạo người dùng admin mới (Chỉ dành cho Super Admin)

Yêu cầu mẫu:
{
  "name": "Admin mới",
  "email": "newadmin@example.com",
  "password": "password123",
  "passwordConfirm": "password123",
  "role": "admin"
}
GET /api/users/admins

Lấy danh sách tất cả admin (Chỉ dành cho Super Admin)

PUT /api/users/admin/:id

Cập nhật thông tin admin (Chỉ dành cho Super Admin)

PATCH /api/users/admin/:id/deactivate

Vô hiệu hóa tài khoản admin (Chỉ dành cho Super Admin)

PATCH /api/users/admin/:id/permissions

Cập nhật quyền cho admin (Chỉ dành cho Super Admin)

Quản lý nhà cung cấp

GET /api/suppliers

Lấy danh sách tất cả nhà cung cấp (Chỉ dành cho Admin)

GET /api/suppliers/active

Lấy danh sách nhà cung cấp đang hoạt động

GET /api/suppliers/:id

Lấy thông tin nhà cung cấp theo ID

GET /api/suppliers/:id/products

Lấy sản phẩm từ một nhà cung cấp

POST /api/suppliers

Tạo nhà cung cấp mới (Chỉ dành cho Admin)

PUT /api/suppliers/:id

Cập nhật thông tin nhà cung cấp (Chỉ dành cho Admin)

DELETE /api/suppliers/:id

Xóa nhà cung cấp (Chỉ dành cho Admin)

Quản lý thuộc tính

GET /api/attributes

Lấy danh sách tất cả thuộc tính

GET /api/attributes/:id

Lấy thông tin thuộc tính theo ID

GET /api/attributes/:id/values

Lấy giá trị của thuộc tính

POST /api/attributes

Tạo thuộc tính mới (Chỉ dành cho Admin)

PUT /api/attributes/:id

Cập nhật thuộc tính (Chỉ dành cho Admin)

DELETE /api/attributes/:id

Xóa thuộc tính (Chỉ dành cho Admin)

POST /api/attributes/:id/values

Thêm giá trị cho thuộc tính (Chỉ dành cho Admin)

DELETE /api/attributes/:id/values/:valueId

Xóa giá trị thuộc tính (Chỉ dành cho Admin)

Bộ sưu tập Postman

Để dễ dàng kiểm thử API, chúng tôi cung cấp bộ sưu tập Postman đầy đủ với tất cả các endpoint.

Bạn có thể tải xuống bộ sưu tập tại /public/postman và nhập vào Postman để bắt đầu kiểm thử.
